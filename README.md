Uma aplicação web feita com React que permite buscar filmes, visualizar detalhes e salvar seus favoritos — tudo consumindo a API do OMDb.

Visualizar detalhes completos de cada filme

Página dedicada para seus favoritos

Navegação dinâmica com React Router

Tecnologias Utilizadas

React (Hooks)

React Router DOM

Context API

Bootstrap 5

OMDb API
